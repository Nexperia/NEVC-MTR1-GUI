# NEVC-MTR1 Motor Driver GUI

Windows GUI for controlling the Nexperia NEVC-MTR1 motor driver board via SCPI over USB-serial (Arduino Leonardo). Built with Rust + iced.

See [PLAN.md](PLAN.md) for the implementation roadmap.

## Build

Requires Rust stable + Windows 10/11.

```powershell
cargo run
cargo build --release
```
